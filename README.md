
# To-Do List Manager Using Python

# Overview

A menu-driven command-line To-Do List Manager developed using Python.

This project allows users to:

* Add tasks
* View tasks
* Mark tasks as completed
* Delete tasks
* Store tasks permanently using JSON files

#Features

* Menu-driven CLI application
* JSON-based data storage
* Task completion tracking
* Error handling
* Clean and readable code

#Technologies Used

* Python
* JSON
* File Handling
* Functions
* Exception Handling

# Learning Outcomes

* Python Programming
* File Handling
* JSON Data Management
* Function-Based Programming
* Error Handling
